Data Privacy & Security
Here is exactly what happens to your data when you use this tool:
Stays localAccount data & API key — saved only in your browser's localStorage. Never transmitted to any server other than when generating an assessment.
Sent to GoogleForm field contents — your usage notes, engagement details, feedback and challenges are sent to Google's Gemini API only when you click Generate Assessment. Your API key travels as a URL parameter in that same request.
Not involvedNo backend, no Anthropic servers — this app runs entirely in your browser. Anthropic's Claude was used to build it, but plays no role when you use it.
⚠ If your accounts contain sensitive or confidential customer data, check your organisation's policies on Google's Gemini API data usage and ensure you have an appropriate data processing agreement in place.
